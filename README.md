
# ✅ VeriVesion – Fraud-Proof Attendance Tracker

An **AI-powered attendance system** that prevents proxy and fraudulent attendance using **face recognition with liveness (blink) detection**, while ensuring **user privacy and secure cloud storage**.

---

## 📌 Problem Statement  
**Open Innovation – Fraud-Proof Attendance Tracking**

Traditional attendance systems are prone to:
- Proxy attendance using photos or videos
- Manual errors and impersonation
- Lack of privacy and data security
- Poor scalability

---

## 💡 Solution Overview

**VeriVesion** is an intelligent, automated attendance system that:
- Verifies **real human presence** using blink-based liveness detection
- Uses **face recognition** to identify registered users
- Blurs background automatically to protect privacy
- Stores verified attendance data securely in **Firebase**
- Marks attendance only after successful identity + liveness verification

---

## 🚀 How It Works

1. Captures the user’s face in real time  
2. Prompts the user to **blink** to confirm live presence  
3. Matches the face with registered user data  
4. Blurs the background for privacy protection  
5. Marks attendance only if:
   - Face recognition ✅  
   - Blink detection ✅  
6. Stores attendance securely with **user ID & timestamp** in Firebase  

---

## ✨ Key Features

- 👤 **Face Recognition Based Identification**
- 👁️ **Liveness Detection using Blink Verification**
- ❌ **Prevents Photo & Video-Based Fraud**
- 🛡️ **Background Blurring for Privacy**
- ⚡ **Real-Time Attendance Processing**
- ☁️ **Secure Cloud Storage with Firebase**
- 🔐 **Authentication & User Validation**
- 📈 **Scalable for Institutions, Offices & Classrooms**

---

## 🆚 Why VeriVesion is Different

- Uses **liveness detection**, unlike basic face-recognition systems
- Prevents:
  - Photo-based proxy attendance
  - Video replay attacks
- Privacy-first design with **automatic background blurring**
- Cloud-based secure storage instead of local/manual records
- Administrator-only access to sensitive attendance data
